# DSA Visualizer
- This repo will help in visualizing complex data structure algorithms
- This is developed based on the my visualizing capabilities.
  
## Features
- Interactive visualization
- Light/ Dark theme

## Contributions
- To contribute fork the repository
- Clone the repository
  ```sh
  git clone https://github.com/MiriyalaJayanth-19/DSA_Visualizer.git
  ```
- Add the Algorithm script to
  ```\scripts```
- Modify registry.js and index.html
